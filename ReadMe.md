# 토이 프로젝트

## Contents 
- ### 실행 동영상

  - 준비중입니다.

- ### 설명

  - mvp, dagger2, rx, room, kotlin을 적용한 토이 프로젝트 입니다.(rx등 추가 작업 예정)

  - 실행 시키시려면 themoviedb 에서 apikey 발급 후 메인 엑티비의 apikey에 값을 입력해주셔야 합니다.(별도로 ignore처리하지 않았습니다.)


## 진행중 어려웠던 점과 이를 극복한 방법

  -dagger의 ContributesAndroidInjector 와 Binds 그리고 SubComponent(ContributesAndroidInjector로 대체) 같은 개념이 부족하여 dagger를 적용하는데 애를 먹었는데 각종 블로그와 유튜브 그리고 googlesample의 예제 등을 통해 이를 해결할 수 있었습니다.
